class DaneOsobowe {

   //Utworzenie zmiennych
    public String imie;
    private String nazwisko;
    protected String klasa;
    public DaneOsobowe(String imie){
       this.imie = imie;
   }
}

class WypisanieImienia {
    DaneOsobowe ds = new DaneOsobowe("wojciech")
   //Wypisanie imienia
    System.out.println(ds.imie);
}

class WypisanieKlasy extends DaneOsobowe{
   //Wypisanie Klasy
    System.out.println(klasa);
}
Utworzenie zmiennych:
Dzięki modyfikatorom dostępu takimi jak public, private i protected możemy manipulować widocznością zmiennych.

Dzięki public możemy użyć zmiennych poza klasą (Wypisanie Imienia), z kolei dzięki protected możemy użyć podczas dziedziczenia klasy(Wypisanie Klasy)

Aby posiadać zahermetyzowane zmienne musimy użyć private który zabrania użycia zmiennej poza klasą.
