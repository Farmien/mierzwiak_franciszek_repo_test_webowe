public class Card {
    private int number;
    private String name;

    public Card(int number, String name){
       this.number = multiplyByTen(number);
       this.name = name;
    }
    //Publiczne Metody
    public int getNumber(){
       return number;
    }
    public String getName(){
       return name;
    }
    public void setNumber(int number){
       this.number = number;
    }
    public void setName(String name){
       this.name = name;
    }
    //Prywatne Metody
    private int multiplyByTen(int number){
       return number * 10;
    }
}
public class CardReader {
    Card card = new Card(11,"walet")
    System.out.println(card.getName());
}
Publiczne Metody:
Publicznych metod używamy, kiedy chcemy użyć methody poza klasą.(Wczytanie nazwy karty poza klasą karty)

Prywatne Metody:
Prywatnyh metod używamy, kiedy chcemy użyć metody tylko wewnątrz klasy.(Mnożenie liczby przez 10)
